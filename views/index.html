This is the front end for my Datastructures Exam. It uses some basic HTML and Javascript.

You will use this box to enter and send your numbers to the backend,
where they'll be added to the binary search tree, and returned back
to you in the box to the right.


Your query should look like this : 40,23,21,4,5,51,32,1...etc

The get request from the database will show up here.
